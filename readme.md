Analyse de Données - Speed Dating Project
Ce projet propose une exploration approfondie d'un jeu de données issu d'événements de "Speed Dating". L'objectif est de comprendre les mécanismes de décision lors de ces rencontres rapides et d'identifier les facteurs qui influencent la création d'un "match" (une attirance mutuelle).

Présentation du Projet
Le projet analyse les comportements des participants, leurs préférences déclarées par rapport à leurs choix réels, et l'impact de divers critères (âge, origine, centres d'intérêt, etc.) sur le succès des rencontres.

Technologies Utilisées
Le projet est développé en Python au sein d'un notebook Jupyter et utilise les bibliothèques suivantes :

Pandas & Numpy : Pour la manipulation et le nettoyage des données.

Matplotlib & Seaborn : Pour les visualisations statistiques.

Plotly Express : Pour des graphiques interactifs.

Scipy & Statsmodels : Pour les analyses statistiques et probabilistes.

Analyses Principales
Le notebook explore plusieurs dimensions :

Prétraitement des données : Nettoyage et préparation des variables.

Exploration Statistique : Analyse de la distribution des participants (âge, profession, origine).

Facteurs de décision : Étude de l'importance accordée à l'attirance, l'intelligence, la sincérité, l'humour et l'ambition.

Corrélations : Analyse des liens entre les intérêts communs et la probabilité d'un match.

Conclusions et Recommandations
D'après les analyses effectuées dans le notebook :

Durée des rencontres : Les participants ont souvent trouvé les rencontres trop courtes. Une recommandation est d'allonger le temps d'échange tout en limitant peut-être le nombre total de rencontres.

Perception de soi : Les participants ont tendance à être orgueilleux sur le moment, devenant légèrement plus modestes le lendemain de l'événement.

L'expérience compte : Plus un participant a d'expérience dans ces soirées, plus son taux de match a tendance à augmenter, suggérant un effet d'apprentissage ou d'aisance acquise.
